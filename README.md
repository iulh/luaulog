## Installation

Using [**Wally**](https://wally.run/):
```bash
log = "realthx1/luaulog@0.2.0"
```
Using [**pesde**](https://pesde.dev/):
```bash
pesde add wally#realthx1/luaulog@0.2.0 -a log -t roblox
```
